# Vite 自定义插件

一个为 Vite 提供的自定义插件集合，包含多种实用功能，帮助开发者更高效地使用 Vite 构建项目。

## 功能特性

- **copyFile**：复制文件或目录到指定位置
- **injectIco**：注入网站图标链接到 HTML 文件的头部
- **易于使用**：简单的配置即可使用所有功能
- **高度可配置**：支持多种配置选项，满足不同需求
- **类型安全**：完整的 TypeScript 类型支持

## 快速开始

请查看 [快速开始](./guide/quick-start) 指南，了解如何安装和使用这些插件。
