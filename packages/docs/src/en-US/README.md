# Vite Custom Plugins

A collection of custom plugins for Vite, providing various useful features to help developers build projects more efficiently with Vite.

## Features

- **copyFile**: Copy files or directories to specified locations
- **injectIco**: Inject website icon links into the head of HTML files
- **Easy to use**: Simple configuration to use all features
- **Highly configurable**: Support multiple configuration options to meet different needs
- **Type safe**: Complete TypeScript type support
