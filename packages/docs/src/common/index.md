# common

公共工具模块，提供文件系统操作、格式化、对象处理、参数验证等通用功能。

```typescript
import { ... } from '@meng-xi/vite-plugin/common'
```

## 模块列表

| 模块                       | 说明             |
| -------------------------- | ---------------- |
| [fs](./fs)                 | 文件系统操作工具 |
| [format](./format)         | 格式化工具       |
| [object](./object)         | 对象处理工具     |
| [validation](./validation) | 参数验证器       |
